# Branding Specification: Agentic RAGFlow Studio

## Requirement: 品牌名称一致性

### Scenario: 所有用户可见的品牌名

- **GIVEN** 系统中存在品牌名引用
- **THEN** 全名使用 "Agentic RAGFlow Studio"
- **AND** 技术标识使用 "agentic-ragflow-studio"
- **AND** UI 简称使用 "Agentic RAGFlow" (Sider) 或 "AR" (折叠态)
- **AND** `openspec/changes/archive/` 中的历史引用保持不变
